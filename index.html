Concept pour l'architecture des projets Tailwind

Du token au css injectée

  • @theme { ... } : C'est le dictionnaire de données (JavaScript/Tokens). Il est lu par le compilateur pour définir les valeurs de votre Design System.
  • @layer utilities : C'est le résultat final de l'injection du code CSS généré par le compilateur en utilisant les valeurs du theme.

1-Couche @layer base
Rôle : La Fondation / Réinitialisation.
Prio : La plus basse. Permet à tout le reste d'écraser facilement les styles par défaut du navigateur.

2-Couche @layer components
Rôle : L'Assemblage de Composants (via @apply).
Prio : Intermédiaire. Peut être écrasée par la couche supérieure.

3-Couche @layer utilities
Rôle : Le Cœur Atomique.
Prio : La plus élevée. Garantit que les utilitaires appliqués dans le HTML priment sur tout.
