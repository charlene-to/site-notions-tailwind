Dégradés

Background-Image, -size, -attachment, -position, -repeat

  • Image via le dossier img
  • Size : bg-auto / cover / contain
  • Attachement : bg-fixed
  • Position : bg-bottom / center / left / right / top / right-bottom / left-top
  • Repeat : bg-repeat / no-repeat

La classe bg-cover applique la propriété CSS background-size: cover;.
Rôle : L'image s'agrandit pour couvrir entièrement l'arrière-plan de la div. ex : hero header


La classe bg-contain applique la propriété CSS background-size: contain;.
Rôle : L'image sera toujours visible en entier. Cependant, si le ratio d'aspect de l'image ne correspond pas à celui de la div, cela laissera des bandes vides (marges). ex : logo

Opacité réduite

opacity-50 rend l'élément semi-transparent (50% opaque, 50% transparent).
opacity: 0 rend l'élément complètement invisible (transparent).
opacity: 1 ou opacity-100 rend l'élément complètement opaque (non transparent).
