2-Visibilité

Les tailles à connaître

  • ex : width 50 : w-50
  • Rappel : 50 correspond à 200px
  • centrage automatique : w-auto
  • moitié du parent : w-1/2, w-1/3 (tiers) etc
  • w-full : 100% de son conteneur parent (width: 100%)
  • w-screen, h-screen : 100% de la largeur/hauteur de l'écran (viewport) (width: 100vw)
  • min-w-0 : Définit la largeur minimale à 0 (souvent utilisé pour permettre à un élément de se réduire au maximum )
  • Aller dans personnalisation pour en savoir plus

4-Object-fit / position Image

Différence avec background-image qui est une pure décoration et ignoré pour le référencement

  • Accessible aux lecteurs d'écran (via la balise alt)
  • L'image est scannée comme un contenu essentiel (l'image est là pour l'information)

Ses valeurs

  • étendue dans le conteneur : object-cover / contain / fill / none
  • centre la partie de ma photo : object-bottom / center / left / right / left-bottom / right-top

object-cover :
l'image est coupée sur les bords (soit en haut/bas, soit à gauche/droite) pour s'assurer qu'il n'y ait pas d'espace vide.

object-contain :
laissera des bandes d'espace vide si l'img est trop petite

ATTENTION : pour les exemples suivants : w-full h-full :
Pour que l'image respecte les dimensions de la div parente

object-top :
le haut doit rester visible

object-bottom :
le bas doit rester visible
